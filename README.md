## Snake Game

### Funcionalidades

 - [x] Mover a cobra
 - [x] Sistema de pontos 
 - [x] Aumentar o tamanho da cobra enquanto os pontos aumentar 
 - [ ] Terminar o jogo ao colidir com seu corpo
 - [ ] Atalhos para o jogo (Q para sair, S para mostrar os pontos, etc)

### Bibliotecas utilizadas
 + Pygame
